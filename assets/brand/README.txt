Place your high-res brand assets here:

- icon_1024.png       (1024x1024)   Used for launcher icons & Android 12 icon image
- splash_1024.png     (>=512, ideally 1024px square) Used for native splash branding (bottom)
- wordmark.svg        (vector wordmark for in-app splash)
- sparkle.json        (optional Lottie accent)

After adding files, run:
  flutter pub get
  flutter pub run flutter_launcher_icons
  flutter pub run flutter_native_splash:create
